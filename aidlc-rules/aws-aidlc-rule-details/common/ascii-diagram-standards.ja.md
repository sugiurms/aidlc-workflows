# ASCII 図版標準

## 必須: 基本 ASCII のみを使用すること

**重要**: 図版には必ず基本 ASCII 文字のみを使用してください（最大限の互換性を確保するため）。

### ✅ 許可: `+` `-` `|` `^` `v` `<` `>` および英数字テキスト

### ❌ 禁止: Unicode ボックス描画文字
- 不可: `┌` `─` `│` `└` `┐` `┘` `├` `┤` `┬` `┴` `┼` `▼` `▲` `►` `◄`
- 理由: フォントやプラットフォーム間でレンダリングが一致しないため

## 標準 ASCII 図版パターン

### 重要: 文字幅のルール
**ボックス内のすべての行は、スペースを含めて正確に同じ文字数でなければなりません。**

✅ 正しい例 (すべての行が 67 文字):
```
+---------------------------------------------------------------+
|                      Component Name                           |
|  Description text here                                        |
+---------------------------------------------------------------+
```

❌ 誤った例 (幅が不揃い):
```
+---------------------------------------------------------------+
|                      Component Name                           |
|  Description text here                                   |
+---------------------------------------------------------------+
```

### ボックスパターン
```
+-----------------------------------------------------+
|                                                     |
|              Calculator Application                 |
|                                                     |
|  Provides basic arithmetic operations for users     |
|  through a web-based interface                      |
|                                                     |
+-----------------------------------------------------+
```

### ネストされたボックス
```
+-------------------------------------------------------+
|              Web Server (PHP Runtime)                 |
|  +-------------------------------------------------+  |
|  |  index.php (Monolithic Application)             |  |
|  |  +-------------------------------------------+  |  |
|  |  |  HTML Template (View Layer)               |  |  |
|  |  |  - Form rendering                         |  |  |
|  |  |  - Result display                         |  |  |
|  |  +-------------------------------------------+  |  |
|  +-------------------------------------------------+  |
+-------------------------------------------------------+
```

### 矢印と接続線
```
+----------+
|  Source  |
+----------+
     |
     | HTTP POST
     v
+----------+
|  Target  |
+----------+
```

### 水平フロー
```
+-------+     +-------+     +-------+
| Step1 | --> | Step2 | --> | Step3 |
+-------+     +-------+     +-------+
```

### ラベル付き垂直フロー
```
User Action Flow:
    |
    v
+----------+
|  Input   |
+----------+
    |
    | validates
    v
+----------+
| Process  |
+----------+
    |
    | returns
    v
+----------+
|  Output  |
+----------+
```

## 検証

図版を作成する前に確認してください:
- [ ] 基本 ASCII のみ: `+` `-` `|` `^` `v` `<` `>`
- [ ] Unicode ボックス描画文字なし
- [ ] 位置揃えにはスペース（タブ不可）を使用
- [ ] コーナーには `+` を使用
- [ ] **すべてのボックスの行が同じ文字幅** (スペースを含めて文字数を数えること)
- [ ] テスト: 等幅フォントでコーナーが垂直に揃っていることを確認

## 代替手段

複雑な図版には Mermaid を使用してください（`content-validation.md` 参照）。
