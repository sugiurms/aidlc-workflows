# 運用

**目的**: 将来の運用フェーズ（デプロイ、モニタリング、保守）のプレースホルダー

**ステータス**: このフェーズは現在プレースホルダーで、将来のバージョンで拡張される予定です。

## 将来のスコープ

運用フェーズには最終的に以下が含まれる予定です:
- デプロイの計画と実行
- モニタリングとオブザーバビリティのセットアップ
- インシデント対応手順
- 保守とサポートのワークフロー
- 本番稼働リストの確認

## 現在の状態

すべてのビルドとテストの活動はコンストラクションフェーズに移動されました。
AI-DLC ワークフローは現在、コンストラクション内のビルドとテストフェーズの後に終了します。
